# CSS Tooltip UI

https://roadmap.sh/projects/tooltip-ui

A simple tooltip component built using **HTML** and **CSS**. The tooltip appears above navigation items when the user hovers over them, providing additional context without using JavaScript.

## Features

- Pure HTML and CSS
- Hover-triggered tooltips
- Clean and lightweight design
- Custom tooltip messages for each navigation item
- Easy to customize and extend

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
.
├── tooltip.html
└── tooltip.css
```

## How It Works

Each navigation link contains a hidden tooltip element. When the user hovers over the link, the tooltip becomes visible using CSS hover effects.

## Learning Objectives

This project was created to practice:

- CSS Positioning
- Pseudo-elements
- Hover States
- UI Component Design
- Accessibility with Semantic HTML
